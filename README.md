# MaxArc Auto Analytics Dashboard

## Project Overview
A Power BI dashboard developed to analyze ride-sharing operations, fleet utilization, driver ratings, booking performance, and customer payment behavior.

## Key Metrics
- Total Bookings
- Completed Bookings
- Cancelled Bookings
- Driver Ratings
- Vehicle Performance
- Payment Analysis

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

## Features
- Interactive vehicle selection
- KPI monitoring
- Driver performance analytics
- Payment method analysis
- Location insights
