## 📌 Project Overview

MaxArc Auto Analytics Dashboard is an interactive Power BI solution designed to monitor ride-sharing operations, fleet performance, booking trends, driver ratings, and customer payment behavior.

The dashboard transforms raw operational data into actionable business insights through dynamic visualizations and KPI monitoring.

---

## 🎯 Business Objectives

- Track booking performance
- Monitor fleet utilization
- Analyze driver ratings
- Understand customer payment preferences
- Identify location-wise booking trends
- Improve operational decision-making

---

## 📊 Dashboard Features

✅ Interactive Vehicle Selection

✅ Dynamic KPI Cards

✅ Driver Ratings Analysis

✅ Payment Method Insights

✅ Location-Based Analytics

✅ Booking Trend Monitoring
